# Game of Life

Сделал от великой скуки...

- Правила игры стандартные без модификаций;
- При запуске можно задать количество ячеек под клетки и размер ячейки (SIZE x SIZE);
- Поле можно претаскивать с помощью зажатой правой клавиши мышки;
- Поле можно перемещать при помощи стрелочек на 1 пиксель;
- Масштабирование при помощи колесика.


Горячие клавиши:

F8    - Заполнить поле случайно

F9    - Изменить стиль сетки (нет сетки / зазор между ячейками / сетка включена)

F12   - Создать screenshot.bmp всего поля (а также при нажатии TAB)

PAUSE - Пауза (доступно и при нажатии на SPACE)

\+	  - Ускорение игры (время между шагами уменьшается)

\-     - Замедление игры
